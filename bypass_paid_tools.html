MITMproxy Setup Guide

This guide will help you set up MITMproxy to intercept and modify HTTP traffic, specifically to inject custom content into responses from a target URL.

বাংলা নির্দেশনা: এই গাইডটি আপনাকে MITMproxy সেট আপ করতে সাহায্য করবে, যা HTTP ট্রাফিক ইন্টারসেপ্ট এবং মডিফাই করতে ব্যবহৃত হয়। বিশেষভাবে, টার্গেট URL থেকে আসা রেসপন্সে কাস্টম কন্টেন্ট ইনজেক্ট করার জন্য এটি ব্যবহার করা যাবে।

1Track curl/wget Requests

Use this command to monitor for curl or wget requests in a script:

strace -s 4096 -f -e execve ./chapeey.sh 2>&1 | grep -E "curl|wget"
স্ক্রিপ্টে curl বা wget রিকোয়েস্ট মনিটর করতে এই কমান্ডটি ব্যবহার করুন। এটি আপনাকে দেখাবে স্ক্রিপ্টটি কোন URLs থেকে ডেটা ডাউনলোড করছে।

2Install MITMproxy

Install mitmproxy on your server using the following commands:

sudo apt update
sudo apt install -y mitmproxy
MITMproxy ইনস্টল করতে উপরের কমান্ডগুলো ব্যবহার করুন। প্রথমে প্যাকেজ লিস্ট আপডেট করুন, তারপর mitmproxy ইনস্টল করুন।

3Create Python Script for Response Modification

Create a Python script that will intercept responses and add your custom text:

#!/usr/bin/env python3
# Code from Jubair bro
from mitmproxy import http

TARGET_URL = "https://raw.githubusercontent.com/Amchapeey/strategic/refs/heads/main/keygen"
CUSTOM_LINE = "### Jubair-bro 2027-05-14 94.237.66.145"

def response(flow: http.HTTPFlow):
    if flow.request.pretty_url == TARGET_URL:
        content = flow.response.content.decode()
        if CUSTOM_LINE not in content:
            content += f"\n{CUSTOM_LINE}"
            flow.response.text = content

Save this script as /root/bypass.py

এই পাইথন স্ক্রিপ্টটি টার্গেট URL থেকে আসা রেসপন্সে আপনার কাস্টম টেক্সট যোগ করবে। স্ক্রিপ্টটি /root/bypass.py পাথে সেভ করুন।

4Make Script Executable

Give execute permission to your Python script:

sudo chmod +x /root/bypass.py
স্ক্রিপ্টটি executable করার অনুমতি দিন, যাতে এটি রান করা যায়।

5Create Systemd Service

Create a systemd service to run mitmdump automatically in the background:

sudo nano /etc/systemd/system/bypass.service

Add the following content to the service file:

[Unit]
Description=mitmdump url redirect service on port 8890
After=network.target

[Service]
ExecStart=/usr/bin/mitmdump -s /root/bypass.py -p 8890
Restart=always
User=root
Environment=HTTP_PROXY=http://127.0.0.1:8890
Environment=HTTPS_PROXY=http://127.0.0.1:8890

[Install]
WantedBy=multi-user.target

Note: The service file should be named bypass.service as shown in the original instructions.

Systemd সার্ভিস তৈরি করুন যাতে mitmdump স্বয়ংক্রিয়ভাবে ব্যাকগ্রাউন্ডে চলতে থাকে। ফাইলটির নাম bypass.service রাখুন।

6Enable and Start the Service

Enable and start your mitmdump service with these commands:

sudo systemctl daemon-reload
sudo systemctl enable --now bypass.service
sudo systemctl start bypass.service
sudo systemctl status bypass.service
সার্ভিসটি enable এবং start করুন। শেষ কমান্ডটি সার্ভিসের status চেক করবে - নিশ্চিত হোন যে এটি সক্রিয় এবং কোনো error দেখাচ্ছে না।

7Set Terminal Proxy

Configure your terminal to use the proxy (note the correct port is 8890, 

export http_proxy=http://127.0.0.1:8890
export https_proxy=http://127.0.0.1:8890

Note: The port 8890.

টার্মিনালকে proxy ব্যবহার করার জন্য কনফিগার করুন। মনোযোগ দিন: port 8890 ব্যবহার করুন।

8Test the Setup

Verify that your custom text injection is working:

curl -s https://raw.githubusercontent.com/Amchapeey/strategic/refs/heads/main/keygen
সেটআপটি কাজ করছে কিনা তা verify করুন। এই curl কমান্ডটি রান করলে আপনার যোগ করা কাস্টম লাইনটি আউটপুটে দেখাতে হবে।

9Fix SSL Certificate Issues

If you encounter SSL problems, install the mitmproxy CA certificate:

mkdir -p ~/.mitmproxy
mitmproxy --quit
ls ~/.mitmproxy/
sudo cp ~/.mitmproxy/mitmproxy-ca-cert.pem /usr/local/share/ca-certificates/mitmproxy-ca.crt
sudo update-ca-certificates

Now done ✅✅✅✅✅✅

যদি SSL certificate-related সমস্যা দেখা দেয়, তাহলে mitmproxy-এর CA certificate ইন্সটল করুন। এটি HTTPS ট্রাফিক intercept করতে সাহায্য করবে।
